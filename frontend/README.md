 Project Description
 Customer Relationship Management (CRM) application built using the MERN Stack.

Users can:
- Register
- Login with JWT Authentication
- Add Customers
- View Customers
- Update Customers
- Delete Customers

---

 Tech Stack
- MongoDB
- Express.js
- React.js (Vite)
- Node.js
- Tailwind CSS
- JWT Authentication
- Bcrypt for password hashing

- Project Structure

backend/
- server.js
- routes/
- controllers/
- models/
- middleware/


frontend/
- components/
- pages/
- services/

---

Features
- Secure Authentication using JWT
- Password Hashing using Bcrypt
- Protected Routes using Middleware
- RESTful API
- User-based data filtering
